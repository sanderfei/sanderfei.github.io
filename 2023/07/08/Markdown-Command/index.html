标题：在单词前面添加井号（#）中间加一个空格，#的数量代表了标题的级别。

这是一个三级标题

段落：使用空白行将一行或者多行文本进行分隔

I really like using Markdown.

I think I’ll use it to format all of my documents from now on.

换行：在一行的末尾添加两个或多个空格，然后按回车键，即可创建一个换行。

First line with two spaces after.
And the next line.

强调：

粗体：在单词前后各添加两个星号

I just love bold text.

斜体: 在单词前后各添加一个星号或者下划线

I just love bold text.
I just love bold text.

粗体+斜体：在单词前后各添加三个星号或者下划线

I just love bold text.

引用：要创建块引用，请在段落前添加一个 > 符号。引用可与部分其他Markdown语法嵌套使用。

这是一个引用

引用包含一个段落

引用包含一个段落

嵌套块引用：在要嵌套的块引用前添加一个 >> 符号

嵌套引用

列表

有序列表：在每个列表项前添加数字并紧跟一个英文句点。支持嵌套

  1. First

  2. Second

  3. Third

  4. First

    1. 1.1
    2. 1.2

  5. Second

  6. Third

无序列表：在每个列表项前面添加破折号 (-)、星号 (*) 或加号 (+)

  • First

  • Second

  • Third

  • First

    • First
    • Second

  • Second

  • Third

代码语法：要将单词或短语表示为代码，请将其包裹在反引号 (`) 中

At the command prompt, type nano.

Use `code` in your Markdown file.

分割线语法：要创建分隔线，请在单独一行上使用三个或多个星号 (***)、破折号 (—) 或下划线 (___) ，并且不能包含其他内容。

为了兼容性请在分割线前后均添加空白行

为了兼容性请在分割线前后均添加空白行

链接语法：链接文本放在中括号内，链接地址放在后面的括号中，链接title可选

不同的 Markdown 应用程序处理URL中间的空格方式不一样。为了兼容性，请尽量使用%20代替空格。
  • 链接普通用法
    • 这是一个链接
  • 链接title是当鼠标悬停在链接上时会出现的文字，这个title是可选的，它放在圆括号中链接地址后面，跟链接地址之间以空格分隔。
    • 这是一个有title的链接
  • 使用尖括号可以很方便地把URL或者email地址变成可点击的链接

    • https://sanderfei.github.io/

  • 带格式化的链接
    • 强调链接，在链接的语法前后加星号
      • 我的博客地址：**sander**
    • 链接显示为代码
      • 我的博客地址：sander
  • 引用类型链接：它使URL在Markdown中更易于显示和阅读

图片语法：要添加图像，请使用感叹号 (!), 然后在方括号增加替代文本，图片链接放在圆括号里，括号里的链接后可以增加一个可选的图片标题文本

  • ![图片](../a.jpg "title")

  • 图片和链接嵌套：即给图片添加链接 [![图片](../a.jpg)](www.xxx.com)

转译字符

在 HTML 文件中，有两个字符需要特殊处理： < 和 & 。 < 符号用于起始标签，& 符号则用于标记 HTML 实体，如果你只是想要使用这些符号，你必须要使用实体的形式，像是 < 和 &
Markdown 允许你直接使用这些符号，它帮你自动转义字符。
